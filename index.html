Enterprise Application Integration mit Apache Camel

Was ist Apache Camel?

Mit Apache Camel steht für die Integration von Altsystemen und exotischen Schnittstellen mit einem Zoo von Protokollen, Endpunkten und deren Verfügbarkeit ein leichtgewichtiges Enterprise Application Integration Framework zur Verfügung, welches nicht nur über 200 Integrationen out-of-the-box bereitstellt, sondern auch anspruchsvolle Integrationsszenarien mit wenig Aufwand realisierbar macht.
